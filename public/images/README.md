# Image Setup Instructions

Please save the following images to this directory (`public/images/`):

## Required Images:

1. **hero-bg.jpg** - The first image (detective/investigator with police station background)
   - Use the dark blue atmospheric image with the silhouette
   - This will be used as the hero section background

2. **documents-investigation.jpg** - The documents with magnifying glass image
   - Use the image showing documents and magnifying glass
   - This will be used in the "Who We Are" section background

3. **motor-investigation.jpg** - The car accident investigation image
   - Use the image showing the investigator examining a car with police lights
   - This will be used in the "Core Specializations" section background

4. **contact-background.jpg** - The business card with map image
   - Use the image showing the Gagandeep Singh & Associates business card with the glowing map
   - This will be used in the Contact page hero section

## How to Save:

1. Save each image with the exact filename listed above
2. Place them in this directory: `public/images/`
3. The images should be in JPG format
4. Recommended size: 1920x1080 or larger for best quality

## Current Status:

The CSS has been updated to reference these images. Once you save the images with the correct filenames, they will automatically appear on the website.
